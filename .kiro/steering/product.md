# 产品概述

Claude 对话时间轴扩展是一个为 Chrome/Edge 浏览器设计的扩展程序，为 Claude.ai 对话页面添加交互式时间轴导航侧边栏。

## 核心功能

- 交互式时间轴，每条用户消息对应一个可点击的节点
- 对话进行中实时更新
- 悬停显示消息摘要预览
- 长按标记/收藏重要消息（按对话 ID 持久化保存）
- 自动适配主题（浅色/深色模式）
- 平滑滚动导航
- 鱼眼模式（用于长对话，展开焦点区域，聚合远处消息）

## 目标平台

- Claude.ai 网页界面 (https://claude.ai/*)
- Chrome 和 Edge 浏览器（Manifest V3）
