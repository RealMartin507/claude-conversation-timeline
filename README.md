# 🕰 Claude Conversation Timeline Extension

为 Claude.ai 的对话页面添加交互式时间轴导航功能。

此扩展会在网页右侧添加一个优雅的悬浮时间轴，帮助你快速了解长对话的结构，并一键跳转到任意一条用户消息。

## ✨ 功能特点

- **📍 快速跳转**：每个用户提问都会在时间轴上生成一个交互节点，点击即可平滑滚动到对应位置。
- **📝 悬停预览**：鼠标悬停在节点上，自动显示该条消息的摘要内容，无需跳转即可快速浏览上下文。
- **⭐ 标记重点**：**长按**时间轴上的节点，可以将其标记为“星标/重点”，节点会变为醒目的颜色。星标状态会根据对话 ID 自动保存，下次刷新依然存在。
- **🌗 自动主题适配**：完美支持 Claude 的浅色（Light）和深色（Dark）模式。
    - **浅色模式**：清新的磨砂玻璃风格。
    - **深色模式**：极简的“悬浮星点”设计，背景隐形，仅保留节点，极致沉浸，不干扰阅读。
- **🖱 智能交互**：支持点击、长按、悬停等多种交互方式，操作流畅自然。

## 🧩 如何安装 (Chrome / Edge)

### 🛠 手动安装 (开发者模式)

本插件目前支持通过加载解压扩展程序的方式安装：

1.  下载本项目源代码到本地计算机。
2.  在 Chrome 或 Edge 浏览器地址栏输入：`chrome://extensions/` 并回车进入扩展管理页面。
3.  打开右上角的 **"开发者模式" (Developer mode)** 开关。
4.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)** 按钮。
5.  选择本项目所在的文件夹（即包含 `manifest.json` 的文件夹）。
6.  安装完成！

现在，打开任意一个 [Claude.ai](https://claude.ai) 的对话页面，右侧即会出现时间轴导航条。

## ⚙️ 使用说明

- **点击节点**：滚动到该条消息。
- **长按节点**：标记/取消标记为“星标”。
- **悬停节点**：查看消息摘要。
